# Matchamatch

## Development

```bash
npm install
npm run dev
```

## Tests

```bash
npm run test:core
npm run test:e2e
```

## Web Build

```bash
npm run build:web
```

## Mobile Shell

```bash
npm run mobile:sync
npm run android --workspace @matchamatch/mobile-shell
npm run ios --workspace @matchamatch/mobile-shell
```
